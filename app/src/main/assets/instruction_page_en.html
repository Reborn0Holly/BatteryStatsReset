This guide assumes you already have Shizuku installed, otherwise it won't work!

Step 1: Download and move the file factory_ftl_default.xml to the root of the file system.

Step 2: Launch Battery Stats Reset, grant file read and write permissions, and enable the Shizuku service.

Step 3: Now press "Install", wait for the installation to complete, and launch the app.

Step 4: Follow the instructions on the screen. If the xml file was moved correctly, everything should work.

Step 5: After completing the process, you can reboot and delete the installed files if desired.
